# Sonocea Brand Portal (Replica)

Local UI scaffold that reproduces the overall look/structure of the Sonocea brand portal:
clean off-white canvas, Aeonik typography, sticky index nav, and sectioned content blocks.

## Run locally

```bash
npm install
npm run dev
```

Then open the URL shown in the terminal (usually `http://localhost:5173`).

## Font

This project loads `AeonikPro-Regular.otf` from the project root via `@font-face`.
